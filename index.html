This website is so swaggy 🤯

Xan.lol was originally made with no-code site building service Carrd.co, but after having a good time paying for its top teir and not coding much myself, I decided to export my sites. The source code of this one is now available on GitHub (under a Mozilla Public License and Creative Commons BY-NC-SA) and is hosted with Cloudflare Pages. I use Contabo to host my own Coolify instance, which is connected to a few xan.lol subdomains. I often think about updating this main site to have more content in the form of a microblog (&/or a full blog), integrations with services I use (Last.fm, Discord, etc.), lists of my favorite things, and a better layout.
